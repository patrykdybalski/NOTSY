<p id="description">The NOTSY project is a fast lightweight tool designed to provide a simple and user-friendly interface for managing plans notes and checking the current weather. Created to enhance skills and build a portfolio the application leverages various Flutter and Firebase technologies to deliver a comprehensive user experience.</p>
